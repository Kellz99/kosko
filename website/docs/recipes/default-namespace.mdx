You can use `@kosko/plugin-set-metadata` to set a default namespace for all Kubernetes manifests.

## Install

<NpmInstallCommand>@kosko/plugin-set-metadata</NpmInstallCommand>

## Configuration

```toml title="kosko.toml"
[[plugins]]
name = "@kosko/plugin-set-metadata"
config.namespace.value = "dev"
```
