# Contributing

## Getting Started

Download dependencies.

```sh
npm install
```

Bootstrap subpackages.

```sh
npm run bootstrap
```

## Developing

Watch changes of TypeScript files and rebuild automatically.

```sh
npm run watch
```

Delete built files.

```sh
npm run clean
```

## Testing

### Unit Tests

Run unit tests.

```sh
npm run test:unit
```

### Integration Tests

Build TypeScript files.

```sh
npm run build
```

Run integration tests.

```sh
npm run test:integration
```

## Linting

Lint TypeScript and JavaScript files.

```
npm run lint
```

## Styleguides

### Git Commit Messages

Please follow [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0-beta.4/).

### Code Formatting

We use [Prettier](https://prettier.io/) to format all the code.
